Azure Linkboard

The Azure Linkboard is a lightweight, cheap to run, and fast bookmark archival site designed to illustrate using Web API and Azure to build the backbone of a real and functional application that supports multiple clients.

If you're reading this then you're using the AngularJS single page application client - coming soon are iOS (Swift based) and Windows 8 Store App clients.

The site can be used here as is however you can also host your own instance of the site and you can get the code below.
